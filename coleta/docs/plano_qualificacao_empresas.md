## Plano de Ação por Empresa

### PRIORIDADE 1 — Empresas grandes com lacunas graves

#### Tenda (469) — Lacunas: dorms, área, UH, vagas, lazer, RI
- **Fonte:** API JSON em `/Empreendimento/FiltroSuperiorAvancado?_estado=XX`
- **O que já temos:** nome, endereço, cidade, estado, bairro, fase, preço, lat/lon
- **O que falta:**
  - `dormitorios_descricao` (49%) — A API retorna `Quartos` mas só como "2", sem detalhe. Precisa visitar página individual para extrair "1 e 2 dorms"
  - `area_min_m2` / `area_max_m2` (0%) — Não vem na API. Extrair da página individual
  - `total_unidades` (0%) — Não vem na API. Extrair da página individual
  - `itens_lazer` (0%) — A API retorna `Facilidades` mas veio vazio. Extrair da página
  - `registro_incorporacao` (0%) — Visitar página e buscar padrão "R.I. nº XXX"
- **Ação:** Criar `enriquecer_tenda.py` que visita cada URL `/apartamentos-a-venda/{uf}/{cidade}/{slug}` e extrai dados adicionais via BeautifulSoup
- **Tempo estimado:** ~15 min (469 páginas × 2s)

## Estratégia de RI (Registro de Incorporação)

O RI é o dado mais difícil de extrair automaticamente. Situação atual:

| Status | Empresas | % do total |
|--------|----------|-----------|
| RI > 50% | Cury (100%), MRV (90%), Plano&Plano (70%), FYP (75%), Sousa Araujo (53%) | ~30% |
| RI 1-50% | Direcional (48%), Econ (11%), ACLF (10%), Mundo Apto (47%), Viva Benx (100%), Vibra (36%), Rev3 (90%) | ~15% |
| RI = 0% | Todas as novas + Tenda, Vitta, EBM, etc. | ~55% |

**Onde extrair RI:**
1. **Páginas dos empreendimentos** — buscar padrões: "R.I. nº", "Registro de Incorporação", "Matrícula nº", "inscrito sob o nº"
2. **Rodapé legal** — muitos sites colocam RI no footer da página do empreendimento
3. **PDFs de book/material** — alguns sites têm PDFs com RI (mais difícil de extrair)

**Ação:** Criar `enriquecer_ri_novos.py` que visita cada URL e busca padrões de RI no texto completo da página
